g Command

A Efficient CMD Tool.

Part1. Supported Commands

 1echo    g command - A Efficient CMD Tool. 
 2echo    ------------------------------------------------------------------------- 
 3echo    cmd                remark
 4echo    ------------------------------------------------------------------------- 
 5echo    g bin              打开目录"D:\Softwares\_Bin"
 6echo    g wsp              打开目录"D:\Workspace"
 7echo    g doc              打开目录"D:\Workspace\Doc"
 8echo    g prg              打开目录"D:\Workspace\Program"
 9echo    g prj              打开目录"D:\Workspace\Project"
10echo    g g                打开文件"D:\Softwares\_Bin\README.md" and "g.bat"
11echo    g wlog             打开文件"D:\Workspace\Doc\Worklog\工作日志_2020.md"
12echo    g md [fileName]    启动程序Typora并打开[fileName]
13echo    g jp               启动程序Jupyter notebook
14echo    g ps               启动程序PhotoShop
15echo    g npp [fileName]   启动程序Filezilla并打开[fileName]
16echo    g ftp              启动程序Filezilla
17echo    g xm               启动程序XMind
18echo    ------------------------------------------------------------------------- 
19echo    enjoy it! ;)

Part2. Use method

  • Download g.bat
  • Edit g.bat according to your requirement.
  • Add the parent path of g.bat to OS Path.
  • Enjoy it ; )
